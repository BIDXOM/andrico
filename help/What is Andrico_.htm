What is Andrico?

  • ANDRICO is a program product, developed for devices based on Android OS, enabling users to download data from Facebook, such as friends contacts(v 1.0), events information and so on(latter versions).

What ANRICO consists of?

  • Core connecting with the Facebook and downloading all the essential information
  • Content Provider storing the information downloaded from Facebook
  • Application displaying, sorting out and filtering downloaded data

Who can use ANDRICO?

  • Owners of Android based devices can use ANDRICO to simplify addition and update of contacts(v1.0), events and other FaceBook information(latter versions)
  • Developers can use the Content Provider and the Core to develop their own applications
