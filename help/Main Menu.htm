Main Menu

Main menu of Andrico looks like this:

  • By clicking VIEW CONTACTS, you'll go to your contact list where you can choose one of your facebook friend and to get information about him or her. If your Contact database is empty, Andrico'll suggest you to create authorization token and synchronize your local contact database with information on a Facebook server
  • By clicking VIEW EVENTS, you'll go to your event list on a Facebook. In that menu you can choose one of the events and get information about it.
  • By clicking SYNCHRONIZE, you'll go to synchronization activity, where you can relogin - if you are another user of Andrico, synchronize with server or edit some settings
  • By clicking ABOUT, you'll see some information about installed program and contact information of developers
